**1\. 商品分类列表接口**

##### 接口功能
> 新增商品分类

##### HTTP请求方式
> POST + JSON

##### URL
> [http://www.xxx.com/smallcashier/category/save](http://www.xxx.com/smallcashier/category/save)

##### 请求参数

|参数|必选|类型|说明|
|:---|:---|:---|:---|
|name|true|string|商品名称|

##### 返回字段

|返回字段|字段类型|说明|
|:---|:---|:---|
|data|string|数据|
|code|int|响应编码|
|msg|string|响应信息|


###### 示例
> 请求
``` json
{
	"name": "1",
	"pageSize": 1,
	"pageNum": 1
}
```
> 成功
``` json
{
    "data": {
      "id": 1,
      "name": "商品分类名称"
    },
    "code": 200,
    "msg": "success"
}
```
> 失败
``` json
{
    "data": null,
    "code": -1,
    "msg": "failed"
}
```
