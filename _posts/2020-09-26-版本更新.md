**1\. 查询应用是否时最新版本**
##### 接口功能
> 查询应用是否时最新版本

##### HTTP请求方式
> POST+JSON

##### URL
> [http://www.xxx.com/smallcashier/transfer/version/get](http://www.xxx.com/smallcashier/version/get)

##### 请求参数

|请求字段|字段类型|说明|
|:---|:---|:---|
|verCode|string|当前应用版本|
|appType|int|win: windows应用. android: android应用|

##### 返回参数

|返回字段|字段类型|说明|
|:---|:---|:---|
|data|object|应用版本信息|
|code|int|响应编码|
|msg|string|响应信息|

#### data字段

|返回字段|字段类型|说明|
|:---|:---|:---|
|verName|string|应用版本|
|verCode|int|应用版本号|
|url|string|应用版本下载地址|
|isNew|boolean|是否时最新版本|
|appType|string|应用类型|
|comment|boolean|备注|

###### 示例
> 请求
``` json
{
  "verCode": 1,
  "appType": "win"
}
```
> 成功
``` json
{
    "data": {},
    "code": 200,
    "msg": "OK"
}
```
> 失败
``` json
{
    "data": null,
    "code": -1,
    "msg": "failed"
}
```
