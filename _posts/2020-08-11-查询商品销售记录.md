**1\. 查询商品销售记录接口**
##### 接口功能
> 查询商品销售记录信息

##### HTTP请求方式
> POST + JSON

##### URL
> [http://www.xxx.com/smallcashier/inventory/detail](http://www.xxx.com/smallcashier/inventory/detail)

##### 请求参数

|参数|必选|类型|说明|
|:---|:---|:---|:---|
|shopId|true|long|店铺id|
|goodsId|true|long|商品id|
|pageNum|true|int|描述|
|pageSize|true|int|分页大小|

###### 单位枚举

|参数|类型|说明|
|:---|:---|:---|
|0|int|公斤|
|1|int|个|
|2|int|箱|

##### 返回data字段

|返回字段|字段类型|说明|
|:---|:---|:---|
|data|string|数据|
|code|int|响应编码|
|msg|string|响应信息|

###### 示例
> 请求
``` json
{
  "shopId": 1,
  "goodsId": 1,
  "pageNum": 1,
  "pageSize": 2
}
```
> 成功
``` json
{
    "data": {
      "employeeId": 111,
      "employeeName": "aaa",
      "dateTime": "2020-08-15",
      "count": 1
    },
    "code": 200,
    "msg": "success"
}
```
> 失败
``` json
{
    "data": null,
    "code": -1,
    "msg": "failed"
}
```
